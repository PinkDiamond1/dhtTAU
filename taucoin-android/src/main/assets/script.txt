# JPEG大都采用yCrCb色彩模型（y表示亮度，Cr红色分量，Cb表示蓝色分量），所以组件数量一般＝3
# 样本就是单个像素的颜色分量，也可理解为一个样本就是一个组件

# 脚本中数字代表的含义
# y  Cr  Cb   Ss Se Ah Al
# X  X   X  : X  X  X  X

# 扫描的进度参数：
# Ss扫描系数开始的索引
# Se扫描系数结束的索引
# Ah零，用于系数的第一次扫描，否则用于先前扫描的Al
# Al逐次逼近扫描低位位置

# 脚本内容，每行代表一次扫描
# AC 非交错式扫描
0 : 0-0,    0, 1 ;
2 : 0-0,    0, 1 ;
1 : 0-0,    0, 1 ;

0:  1-5,    0, 2 ;
2:  1-63,   0, 1 ;
1:  1-63,   0, 1 ;
0:  6-63,   0, 2 ;
0:  1-63,   2, 1 ;

0,1,2:0-0,  1, 0 ;
2 : 1-63,   1, 0 ;
1 : 1-63,   1, 0 ;
0 : 1-63,   1, 0 ;
